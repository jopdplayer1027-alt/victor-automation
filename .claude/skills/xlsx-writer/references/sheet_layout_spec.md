# 시트 레이아웃 스펙

## 열 구조

| 열 | 내용 | 비고 |
|----|------|------|
| A | 구분 (역할 그룹) | 같은 그룹 행끼리 셀 병합 |
| B | 역할명 | 원문 표기 그대로 |
| C | 이름 | 빈 셀 (섭외팀 후처리) |
| D~ | 날짜별 근무시간 | 각 날짜 1열 |
| 마지막 | 연락처 | 빈 셀 (섭외팀 후처리) |

## 행 구조

| 행 | 내용 |
|----|------|
| 1 | 날짜 헤더 (N일, 부가일정 레이블 포함) |
| 2 | 요일 헤더 (월/화/수...) |
| 3+ | 인력 행 (역할별, 인원수만큼) |

## 날짜 열 순서
- 부가 일정 날짜(세팅 등) + 행사 기간 전체 날짜를 오름차순으로 배치
- 비용/단가 열 없음 (재경팀 별도 파일)

## 셀 내 근무시간 형식
- `HH:MM - HH:MM` (예: `09:00 - 22:30`)
- 익일 종료: `HH:MM - 익일 HH:MM` (예: `22:00 - 익일 02:00`)
- 근무 없는 날: 빈 셀

## 역할 그룹 파생 규칙 (create_xlsx.py `derive_group` 함수)
- 팀장/SV/슈퍼 → SV
- 세팅/설치/철수 → 세팅/철수
- 영어 포함 → 영어
- 중국어 포함 → 중국어
- 일반/재고/캐셔/결제 포함 → 일반
- 그 외 → 역할명 그대로

## 색상
- 헤더 행 (날짜/요일): 연파랑 (#D9E1F2)
- 역할 열 (A·B): 연초록 (#E2EFDA)
- 이름·시간·연락처: 흰 배경
