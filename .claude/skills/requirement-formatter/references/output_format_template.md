## 날짜별 인력 스케줄 표

> 인력구분별 행, 날짜별 열 구조.
> 각 셀: 근무시간 (HH:MM - HH:MM) 또는 공란.

```
【 인력 스케줄 】

인력구분              | {date1(요일)} | {date2(요일)} | ...
──────────────────────|───────────────|───────────────|────
{role} ({count}명)    | {start}-{end} | {start}-{end} | ...
...
```

---

## 작성 규칙

1. 날짜는 `M/D(요)` 형식으로 표기 (예: `5/21(목)`)
2. 같은 역할이 여러 명이면 `{role} ({count}명)`으로 표기
3. 해당 날짜에 근무 없으면 셀 공란
4. 역할 순서: 팀장/SV → 외국어스탭 → 일반스탭 → 기타
5. 세팅/철수 등 부가 일정이 있는 날은 날짜 옆에 `[세팅]`, `[철수]` 표시
