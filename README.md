# 🤖 인원요청서 자동 생성 에이전트

이 프로젝트는 영업팀 슬랙 워크플로우에서 올라오는 **"상담하기" 메시지**를 입력받아, 행사 인력 스케줄이 정리된 **표준 인원요청서(Excel 엑셀 파일)** 로 자동 변환해 주는 AI 에이전트 툴입니다. 

섭외팀 직원분들이 일일이 메시지를 읽고 엑셀에 옮겨 적는 수작업 시간을 획기적으로 줄이기 위해 제작되었습니다.

---

## 💡 주요 기능

1. **자동 텍스트 분석 및 정보 추출**
   - 길고 복잡한 슬랙 텍스트 메시지에서 거래처명, 행사명, 날짜, 인력별 스케줄, 부가 일정(세팅 등)을 인공지능이 똑똑하게 읽고 추출해 냅니다.
   - 정보가 누락되었거나 모호한 경우, 임의로 판단하지 않고 사용자에게 명확히 확인하는 기능이 포함되어 있습니다.

2. **엑셀(Excel) 자동 생성**
   - 추출된 데이터를 바탕으로 `output/` 폴더에 자동으로 엑셀(`.xlsx`) 파일을 만들어 줍니다.
   - 직무별(팀장, 스탭, 통역 등), 날짜별로 시트가 알아서 깔끔하게 그려집니다.

3. **수정/변경 사항 자동 감지**
   - 슬랙 메시지에 '수정', '변경' 등의 단어가 포함되어 있으면 수정 요청으로 인식합니다.
   - 기존에 생성된 엑셀 파일이 있다면, 기존 시트 이름 뒤에 `_수정전`을 붙여 보존하고 새로운 시트를 생성하여 데이터 유실을 방지합니다.

---

## 🚀 사용 방법 (진행 예정 사항 포함)

> **안내:** 현재 이 툴은 개발이 진행 중인 상태입니다. 최종 완성 시 아래와 같은 방법으로 사용하실 수 있습니다.

1. 바탕화면 또는 이 폴더에 있는 **`▶ 에이전트 실행.bat`** 파일을 더블 클릭하여 실행합니다.
2. 검은색 터미널(명령 프롬프트) 창이 열리면, 영업팀이 올린 슬랙 메시지를 마우스 우클릭으로 **그대로 붙여넣기** 합니다.
3. 복사가 끝났으면 마지막 줄에 `END` 라고 입력하고 엔터(Enter)를 누릅니다.
4. AI가 잠시 생각한 후, 처리가 완료되면 이 폴더 안의 **`output/`** 폴더에 엑셀 파일(`거래처명_행사명.xlsx`)이 생성됩니다!

---

## 📁 주요 폴더 및 파일 설명

- `▶ 에이전트 실행.bat` : 직원분들이 실제로 클릭해서 실행하는 작동 버튼입니다.
- `output/` : 에이전트가 만들어낸 최종 엑셀 파일들이 저장되는 결과물 폴더입니다.
- `.claude/skills/` : 엑셀을 그리고 메시지를 분석하는 핵심 파이썬 스크립트들이 들어있습니다. (개발용)
- `staffing_request_agent_design.md` : 이 에이전트가 어떻게 동작해야 하는지 규칙이 상세히 적힌 메인 설계도입니다.
